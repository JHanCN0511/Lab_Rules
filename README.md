# 知微驭象实验室

# 研究生培养与管理办法（试行）

# （2026精简版）

` 为保障科研训练质量、实验安全、工作效率和师生之间的长期良性合作，特制定本办法。本办法用于提前明确课题组基本要求和责任边界；

本办法不替代学校、学院及实验平台的正式规定。如本办法与上级规定不一致，以上级规定为准。`

---

## 一、科研态度与实验安全

（1）🙋研究生应以积极主动、认真负责的态度对待科研工作。科研具有探索性和不确定性，实验失败、反复调试均属于正常；应及时记录、分析并沟通，而不是拖延、隐瞒或消极应付。

（2）👏学生应尊重课题组前期投入的研究方案、硬件条件、经费资源和指导时间。遇到问题应先查阅资料、提出初步判断，再与老师或同学讨论。

（3）🔬实验安全是底线要求。开展实验前应完成必要培训，熟悉标准操作流程和风险点；未经培训、未经允许或未掌握操作流程时，不得擅自操作仪器或开展高风险实验。

（3）👨‍🔬使用危险化学品及实验室设备时，应按规定佩戴防护用品，实验结束后及时关闭电源、气源、水源。发现异常应立即停止并报告。

## 二、工作时间、到岗与请假

（1）研究生不是简单意义上的“打卡员工”，但研究生培养需要稳定、持续、可见的时间投入。无课程/考试/学院活动/出差/身体不适等合理原因时，**学生应在工作日按时到教研室/实验室开展科研工作**。

（2）原则上，每天应保证**不少于8小时**有效科研时间，或每周保证**不少于40小时**有效科研时间。（9：00 - 21：00，不含餐点、午休）

（3）课题组不强制无意义加班，不以是否加班作为评价学生的主要标准。但科研工作具有阶段性节点和实验连续性，学生应根据课题需要、个人目标和时间安排进行自主投入，并避免疲劳实验。

（4）因病、家庭事务、课程考试、学院安排、出差或其他合理原因不能到岗，**应提前向指导教师或指定负责人说明**；突发情况可事后及时补充说明。

## 三、跨校区科研工作

（1）考虑到课题组所在学院、实验平台、加工测试条件、公共仪器和合作资源分布在不同校区，跨校区开展实验、测试、讨论和设备协调将不可避免。学生在进入相关课题时应有基本心理准备。

（2）跨校区工作前应提前确认实验目标、设备预约、耗材准备等，尽量减少低效往返。不得长期以“校区距离远”等作为不推进实验工作的常规理由；确有困难应及时提出并共同优化方案。

## 四、日常沟通、周汇报与月度讨论

（1）积极沟通：学生应主动与指导教师沟通科研问题、实验进展、阶段困难和下一步计划。学习压力、生活适应、团队协作和个人发展方面的问题，也欢迎适当沟通。

（2）周报要求：每周五，学生应以PPT形式整理本周工作进展和下一步计划，通过邮件发送给指导教师或按课题组要求提交。

（3）月度汇报：原则上每月至少进行2次线下或线上正式汇报（两周一次）。月度汇报应准备认真、排版规范、数据详实。

（4）文献精读：原则上每月至少进行1次领域文献精读汇报。汇报重点说明论文解决了什么问题、真正创新点在哪里，以及对本人有什么启发。

（5）数据规范：坚持“当周数据当周处理”。应在当周完成初步整理、规范命名、备份和初步分析。有效原始数据不得随意覆盖或删除。

## 五、团队协作、成果意识与附则

（1）团队协作：实验室鼓励互相帮助、跨年级协作和共同成长。合作任务应尽量明确负责人、参与人、时间节点和交付材料；不得只挂名、不投入、不反馈，却在成果、竞赛或奖励中要求同等贡献。

（2）规范署名：论文/专利/软著/竞赛作品/数据集/代码等成果，应以真实贡献为基础进行署名和排序，并遵守学校规定。学生应对自己提交的数据、图表、文字、代码和结论负责。

本办法自发布之日起试行，并可根据课题组发展和学生反馈修订。
